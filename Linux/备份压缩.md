# Linux zip命令
1. 语法\
-l 压缩文件时，把LF字符置换成LF+CR字符
-ll 压缩文件时，把LF+CR字符置换成LF字符
-q 不显示指令执行过程
-r 递归处理，将指定目录下的所有文件和子目录一并处理
2. 将 /home/html/ 这个目录下所有文件和文件夹打包为当前目录下的 html.zip：
> zip -q -r html.zip /home/html
3. 如果在我们在 /home/html 目录下，可以执行以下命令： 
> zip -q -r html.zip *
